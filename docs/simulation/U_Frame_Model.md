# U-Frame Sensor Placement Model

## Objective

Determine optimal sensor node placement for contactless biomass monitoring during truck movement.

---

# Physical System

The monitoring system consists of an inverted aluminium U-frame installed along the truck pathway.

Sensor nodes are mounted along the frame to capture multiple regions of the biomass load.

---

# Initial Sensor Configuration

Proposed arrangement:

              Top Node

                 ↓

        ┌────────────────┐
        │                │
Left →  │    Biomass     │ ← Right
Node    │    Truck       │   Node
        │                │
        └────────────────┘


---

# Parameters Required

## Frame

- Height
- Width
- Structural constraints

## Vehicle

- Maximum truck height
- Maximum truck width
- Biomass loading profile

## Sensor

- Field of View
- Measurement distance
- Mounting angle

---

# Optimization Criteria

Sensor placement will be evaluated based on:

- Coverage area
- Blind regions
- Hardware complexity
- Installation feasibility
